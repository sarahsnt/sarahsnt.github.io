TESLA MODEL X

Informations de base

Les Tesla sont les voitures électriques les plus connues du monde ! Le fondateur de cette marque est Elon Musk. La Tesla model X est une gamme de Tesla très espacée, on y trouve 7 places. Sa particularité se trouve sur les deux portières du coté passagé : les portes s'ouvrent vers le haut ! Sans oublier l'énorme tablette tactile qui envahie la borne; la tablette nous permet de contrôler les fontions de la voiture et nous sert aussi de GPS. Elle nous fait du 0 à 100 km/h entre 2,8 à 4,6 secondes. Vous pouvez vous procurer la Tesla model X à partir de 89 990 euros.
